# Анализ данных о населенных пунктах России

## Задача
Провести комплексное исследование датасета, содержащего информацию о населенных пунктах России. Основные цели включают:

- Предобработку данных: очистку названий от типов населенных пунктов (город, деревня и т.д.) и обработку пропущенных значений.
- Статистический анализ: определение доли различных типов поселений, расчет суммарного населения по категориям.
- Исследование топонимов: поиск самых популярных, коротких и длинных названий, а также анализ частоты употребления определенных прилагательных и начальных букв в именах пунктов.
- Группировку и классификацию: выявление региональных особенностей и поиск необычных названий по тематическим группам.

## Данные
Для проведения анализа данных использовался файл <a href='https://drive.google.com/file/d/1lXS8Xld0Z89e5h0KEb2v_HdFRZTV2F37/view?usp=drive_link' target='_blank'>Russian_ds.tsv</a>. Ниже представлена интерпретация столбцов датафрейма:

- `name:` Полное название населенного пункта с указанием его типа.
- `other_name:` Альтернативное, локальное или историческое название объекта.
- `region_name:` Наименование субъекта Российской Федерации (область, край, республика).
- `district:` Муниципальный район или городской округ.
- `lon:` Географическая долгота центра объекта.
- `lat:` Географическая широта центра объекта.
- `population:` Количество официально зарегистрированных жителей.
- `approx:` Метка, указывающая на точность или приблизительность данных о населении.
